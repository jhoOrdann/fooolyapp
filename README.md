## 🚀 Sobre o projeto

O **Foooly** é um aplicativo de música em que a comunidade contribui fazendo upload de músicas diversas.

---

## ✨ Principais funcionalidades

📝 Letras sincronizadas<br>
🎨 Temas dinâmicos  
📑 Playlists personalizadas  
🔍 Busca por música, artista ou álbum   
📡 Integração com Discord (Rich Presence -- Necessita baixar o app desktop)

---

## 🛠️ Tecnologias utilizadas

- ⚡ **HTML5**
- 🎨 **CSS3**
- 🧠 **JavaScript**
- 🎵 **Web Audio API**
- 🖥️ **Electron** (versão desktop)
- 🔌 Integrações com APIs externas
